# 网页拖拽元素编辑器

一个简单的网页元素拖拽编辑器，允许用户从左侧面板拖拽组件到中间编辑区域，并可以在编辑区域内自由移动组件位置。

![alt 首页](/img/first.png)

## 功能特性

- ✅ 从左侧面板拖拽新组件到中间编辑区域
- ✅ 在中间编辑区域内自由拖拽移动现有组件
- ✅ 组件自动对齐到20px网格
- ✅ 支持多种组件类型（按钮、输入框、图片等）
- ✅ 组件选中和属性编辑
- ✅ 组件删除功能
- ✅ 生成HTML代码

## 如何运行

### 直接运行
直接双击 `index.html` 文件就能查看效果。

### 开发模式（热更新）
建议安装 Vite 并使用 Vite 运行：

```sh
pnpm i

pnpm dev
```

## 如何使用

1. **添加组件**：从左侧面板拖拽任何组件到中间编辑区域
2. **移动组件**：点击选中组件后，可以直接拖拽组件到新的位置
3. **编辑属性**：选中组件后，右侧面板会显示组件属性，可以进行编辑
4. **删除组件**：选中组件后，按 Delete 键或右键点击组件并点击红色删除按钮
5. **查看代码**：点击 "查看代码" 按钮可以查看生成的 HTML 代码
6. **导出代码**：点击 "下载代码" 按钮可以下载生成的 HTML 文件
